# make-advent-calendar-2017

## story

人類が増えすぎたソースコードをスクリプトにビルドさせるようになって、既に半世紀が過ぎていた。エコシステムの周りの巨大なビルドツールは人類の第二の Makefile となり、人々はそこで子を産み、育て、そして死んでいった。
